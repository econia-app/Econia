# 🤖 LES 5 WORKFLOWS COWORK À UTILISER AU QUOTIDIEN

**Comment les utiliser** : tu ouvres Cowork Desktop, tu sélectionnes le Project "Econia Social Media", tu colles le prompt, tu valides. Cowork bosse en autonomie. Tu peux fermer la fenêtre, c'est asynchrone.

---

## ⚡ WORKFLOW 1 — "Prépare ma semaine" (15 min de boulot Cowork)

**Quand l'utiliser** : tous les dimanches soir.

**Prompt à coller** :
```
Prépare ma semaine de contenu social media.

1. Lis le calendrier-editorial.md actuel pour voir où j'en suis
2. Lis 4-analytics/apprentissages.md pour appliquer les leçons précédentes
3. Lis 5-veille/trends-actuelles.md pour intégrer les sons/formats du moment
4. Génère 3 nouveaux scripts complets (format standard) pour mardi, jeudi et samedi
5. Varie les angles éditoriaux (jamais 2 fois le même angle de suite)
6. Sauvegarde-les dans 2-scripts/a-tourner/ avec préfixe YYYY-MM-DD
7. Mets à jour le calendrier-editorial.md
8. Donne-moi un résumé de 5 lignes max : titres + angle + heure de post

Ne me demande pas confirmation, fais tout d'un coup.
```

**Résultat** : tu n'as plus qu'à filmer + monter dans CapCut.

## 📊 WORKFLOW 2 — "Récupère et analyse mes stats" (lundi matin)

**Quand l'utiliser** : tous les lundis matin.

**Prompt à coller** :
```
Récupère mes statistiques de la semaine passée.

1. Ouvre Chrome via Claude in Chrome
2. Va sur tiktok.com/tiktokstudio (je serai déjà connecté)
3. Note pour chaque vidéo des 7 derniers jours : vues, likes, commentaires, partages, watch time moyen, taux de complétion
4. Va sur instagram.com (Insights des Reels)
5. Note les mêmes métriques
6. Met à jour 4-analytics/tiktok-stats.md et instagram-stats.md
7. Identifie : 
   - Ma meilleure vidéo (et pourquoi)
   - Ma pire vidéo (et hypothèse)
   - 3 patterns observés
8. Met à jour 4-analytics/apprentissages.md
9. Fais-moi un résumé de 10 lignes max dans le chat avec recommandations concrètes pour la semaine prochaine
```

**Résultat** : tu sais en 2 min ce qui marche et ce qu'il faut corriger.

## 🔍 WORKFLOW 3 — "Veille trends et nouvelles aides" (1×/2 semaines)

**Quand l'utiliser** : tous les 2 jeudis.

**Prompt à coller** :
```
Fais ma veille bimensuelle.

PARTIE 1 — Trends social media :
1. Recherche web "tendances TikTok France semaine [date du jour]"
2. Recherche web "trends Instagram Reels France 2026"
3. Recherche web "sons viraux TikTok finance France"
4. Identifie 3 nouveaux sons/formats qu'on pourrait utiliser pour Econia
5. Met à jour 5-veille/trends-actuelles.md (garde l'historique, ajoute en haut)

PARTIE 2 — Veille aides et lois :
1. Recherche web "nouvelles aides sociales France [mois en cours] 2026"
2. Recherche web "loi finances 2026 ménages crédit impôt"
3. Recherche web "ameli.fr nouveautés 2026"
4. Identifie tout ce qui pourrait être un sujet de vidéo (changement barème, nouvelle aide, échéance fiscale)
5. Met à jour 5-veille/nouvelles-aides-2026.md

PARTIE 3 — Recommandations :
- Si une grosse news pertinente est sortie cette semaine, propose-moi un script "réaction à chaud" à tourner dans les 24h
- Sinon, suggère-moi quel angle prioriser pour la semaine prochaine

Fais-moi un debrief de 15 lignes max.
```

**Résultat** : tu surfes les tendances + tu réagis vite aux actus = vues garanties.

## 🎬 WORKFLOW 4 — "Génère un script à la volée"

**Quand l'utiliser** : quand tu vois un post qui cartonne et tu veux le transformer en vidéo Econia (réactivité = vues).

**Prompt à coller** :
```
Je viens de voir [DESCRIPTION RAPIDE — ex: "une vidéo TikTok qui fait 2M de vues sur l'inflation"].

Génère-moi un script Econia qui surfe sur cette vague :
- Reprend le format/angle qui a cartonné
- Adapte au sujet "argent perdu / aides non réclamées"
- Format standard du projet
- Tournable et montable en moins de 1h
- Sauvegarde dans 2-scripts/a-tourner/ avec préfixe YYYY-MM-DD-urgent

Donne-moi aussi : le hook exact + la musique recommandée + l'heure idéale de post pour aujourd'hui ou demain.
```

**Résultat** : tu transforme une trend en vidéo Econia en 10 min.

## 📦 WORKFLOW 5 — "Batch mensuel" (1×/mois — gros boost)

**Quand l'utiliser** : 1 dimanche par mois où tu as 3-4h devant toi.

**Prompt à coller** :
```
Prépare-moi 20 scripts d'avance pour le mois.

1. Lis tous les analytics et apprentissages
2. Lis la veille trends et nouvelles aides
3. Génère 20 scripts complets répartis ainsi :
   - 4× format "chiffres choc"
   - 4× format "liste pratique"
   - 3× format "personnage qui parle"
   - 3× format "avant/après cas concret"
   - 2× format "POV anxiogène"
   - 2× format "démythification"
   - 2× format "actu/news"
4. Pour chaque script, propose 3 hooks alternatifs (A/B test)
5. Sauvegarde-les dans 2-scripts/a-tourner/ avec numéro 01 à 20
6. Met à jour le calendrier-editorial.md avec un planning prévisionnel sur 30 jours
7. Crée un fichier 2-scripts/a-tourner/PLAN-MENSUEL.md avec le résumé visuel

Travaille en autonomie complète. Préviens-moi quand c'est fini.
```

**Résultat** : 1 mois de contenu prêt en une seule session. Tu n'as plus qu'à exécuter.

## 💡 ASTUCES POUR MAXIMISER COWORK

### Pair desktop + mobile
Une fois pairé, tu peux **lancer un workflow depuis ton tel** dans le métro et trouver le boulot fait quand tu rentres chez toi. Il faut juste que ton PC reste allumé.

### Custom MCP (avancé, plus tard)
Quand tu seras à l'aise, tu peux ajouter le MCP Notion ou Airtable pour que Cowork track tes performances dans une base de données structurée. Pas urgent.

### Limite de quota
Cowork bouffe les limites. Si tu plafonnes en Pro, soit tu attends le reset, soit tu passes Max (100$/mois) — calcul rentable si tu sors 5+ vidéos/semaine.

### Sécurité
- Ne donne JAMAIS accès à ton dossier `econia/` complet (code Next.js)
- Donne accès UNIQUEMENT à `econia/social-media/`
- Cowork demande confirmation avant de supprimer un fichier — toujours vérifier

### Pour aller plus loin
Quand tu auras 1 000 abonnés, on pourra ajouter :
- Un workflow qui génère les visuels images en bulk (via API d'image)
- Un workflow qui prépare les emails de newsletter pour ta waitlist Supabase
- Un workflow qui surveille les concurrents (autres créateurs finance France)
